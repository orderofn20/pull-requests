Vendor : Datatables

DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, built upon the foundations of progressive enhancement, that adds all of these advanced features to any HTML table.

  • https://datatables.net/
  • License: MIT 

/* 
	:: Vendor Path
	node_modules/datatables.net-*

	:: Installed
	npm install datatables.net-bs4
	npm install datatables.net-autofill-bs4
	npm install datatables.net-buttons-bs4
	npm install datatables.net-colreorder-bs4
	npm install datatables.net-fixedcolumns-bs4
	npm install datatables.net-fixedheader-bs4
	npm install datatables.net-keytable-bs4
	npm install datatables.net-responsive-bs4
	npm install datatables.net-rowgroup-bs4
	npm install datatables.net-rowreorder-bs4
	npm install datatables.net-scroller-bs4
	npm install datatables.net-select-bs4
	npm install jszip

	npm install pdfmake (***** PLEASE READ BELOW *****)

	:: SOW Controller Init
*/	$.SOW.vendor.datatables.init('.table-datatable');
									

@IMPORTANT!
@pdfmake is 2MB MINIFIED!

If PDF export is really needed, pdfmake is loaded by Smarty controller if PDF export option is enabled! 
Smarty is also giving you an option to disable PDF export on mobile devices!

Basic

NOTE: Smarty Datatable using HTML attributes only is curently in Beta!
You can create your own datatable by passing this attribute: data-custom="true"
From this point, you handle the entire script - plugin is loaded as long you keep .table-datatable class!
