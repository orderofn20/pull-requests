File: src/scss/_core/utils/_colors.scss

Text Color : Extended

Size: 7Kb (~3Kb gzipped)
Colors inspired from material design! (text-gray-* already added by bootstrap!)

If not needed, simply remove from: src/scss/main.scss
If backgrounds needed, add the code below to your src/scss/_custom/main.scss and rebuild using GULP. (+8Kb will be added - better use style and hex color) 

// Smarty Extended : Bg Colors [Material Design][8Kb]
@each $color-name, $color-variants in $material-colors {
    @each $color-variant, $hex-value in $color-variants {

        .bg-#{$color-name}-#{$color-variant} { background-color: #{$hex-value}!important; }

    }
}
